# Manual d'Usuari: Dashboard E13BD

Benvingut a la documentació oficial del **Dashboard E13BD**. Aquest manual t'ajudarà a entendre i utilitzar totes les funcionalitats del sistema d'auditoria de bases de dades Oracle.

## Què és el Dashboard E13BD?

És una eina avançada dissenyada per a administradors i auditors de bases de dades que permet:
- **Automatitzar** les comprovacions rutinàries sobre esquemes Oracle.
- **Visualitzar** ràpidament l'estat de salut del sistema.
- **Generar informes** professionals en qüestió de minuts.
- **Utilitzar IA** per resumir i analitzar errors complexos.
