# Post Restart Browser Check

## Goal
Verificar en navegador real que la instancia reiniciada carga shell y rutas activas basicas.

## Tasks
- [ ] Abrir la SPA reiniciada y verificar el shell principal -> Verify: la home responde y aparecen tabs principales.
- [ ] Navegar a Auditoria BBDD y abrir una subpestanya activa -> Verify: la vista carga sin error visible.
- [ ] Cerrar con una comprobacion simple de estabilidad -> Verify: no hay fallo de navegacion ni pantalla en blanco.

## Done When
- [ ] La instancia reiniciada queda verificada en navegador real.
